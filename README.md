🎵 AI‑Powered Playlist Builder
A smart web application that generates personalized playlists using artificial intelligence and exports them directly to Spotify.

🚀 Overview
AI‑Powered Playlist Builder is an intelligent music discovery tool that creates custom playlists based on mood, genre, activity, or listening history.
Using AI‑driven recommendations and seamless Spotify integration, users can generate unique music collections and save them to their Spotify account with a single click.
This project blends AI, modern web development, and music personalization to deliver a fast, intuitive, and creative playlist‑building experience.

✨ Features
- 🎧 AI‑Generated Playlists
Automatically create playlists tailored to the user’s preferences.
- 😌 Mood, Genre, and Activity Filters
Build playlists for studying, working out, relaxing, focusing, or any emotional state.
- 🔄 Optional Listening History Analysis
Improve recommendations by learning from the user’s Spotify habits.
- 🔗 One‑Click Spotify Export
Save the generated playlist directly to the user’s Spotify account.
- ⚡ Fast and Minimal UI
Designed for simplicity, speed, and a smooth user experience.

🛠️ Tech Stack
- Frontend: Angular + TypeScript
- Backend: Node.js / Express 
- AI Layer: Generative models for playlist creation
- Integrations: Spotify Web API
- Styling: CSS Modules

🚀 Descripción
AI‑Powered Playlist Builder es una aplicación web inteligente que crea playlists personalizadas basadas en el estado de ánimo, género musical, actividad o historial de escucha del usuario.
El sistema utiliza modelos de IA para entender preferencias musicales y generar colecciones únicas, listas para escucharse o exportarse directamente a Spotify.
Este proyecto combina experiencia de usuario, machine learning y conectividad con APIs musicales para ofrecer una forma rápida, creativa y automatizada de descubrir música nueva.

✨ Características principales
- 🎧 Playlists generadas por IA
La aplicación analiza preferencias y genera listas únicas en segundos.
- 😌 Basado en mood, género o actividad
Crea playlists para entrenar, estudiar, relajarte, trabajar o cualquier estado emocional.
- 🔄 Aprende del usuario
Opcionalmente, puede utilizar historial de escucha para mejorar recomendaciones.
- 🔗 Integración con Spotify
Exporta la playlist generada directamente a tu cuenta con un solo clic.
- ⚡ Interfaz rápida y minimalista
Diseñada para que crear música sea simple y divertido.

🛠️ Tecnologías utilizadas
- Frontend: Angular / TypeScript
- Backend: Node.js / Express 
- IA: Modelos generativos para análisis y creación de playlists
- Integraciones: Spotify Web API
